# Nocturn Dark — Color Palette

Design reference: **Cursor IDE** dark workbench, with **forest green** accents instead of blue.

| Role | Hex | Usage |
|------|-----|--------|
| Background | `#181818` | Desktop, panels base |
| Background alt | `#1e1e1e` | Windows, editor-like surfaces |
| Surface | `#252526` | Title bars, sidebars, menus |
| Surface alt | `#2d2d2d` | Hover states, elevated UI |
| Border | `#3c3c3c` | Inputs, cards |
| Border subtle | `#2b2b2b` | Window chrome, dividers |
| Foreground | `#cccccc` | Body text |
| Foreground dim | `#858585` | Secondary labels, inactive titles |
| Foreground bright | `#e8e8e8` | Headings, selection text |
| Accent | `#2d6a4f` | Primary buttons, checked switches |
| Accent bright | `#40916c` | Focus rings, active tab underline, window top bar |
| Accent dim | `#1b4332` | Pressed states |
| Accent glow | `#52b788` | Links, switch knobs, highlights |
| Selection | `#264f3f` | List/tree selection |
| Error | `#f48771` | Destructive / close hover |
| Warning | `#cca700` | Caution infobars |
| Success | `#4ade80` | Positive feedback |

## Suggested pairings

- **Icons:** Papirus-Dark, Adwaita, or Tela-circle-dracula-green
- **Cursor:** Bibata-Modern-Ice or a simple dark pointer
- **Terminal:** match `#181818` background with `#52b788` prompt accent
