The Nocturn look

Every hex maps to a role in the GTK theme — background, surface, accent, selection. Same language on the website and on your desktop.

Preview the theme in a browser before you install, or open the live mock in themes/Nocturn/preview.html after cloning the repository.

XFCE Theme Forge

Generate your own variant from a JSON config or the local web UI — same engine that builds Nocturn.

# Windows — design in the browser python tools/xfce-theme-forge/theme-forge.py serve # → http://127.0.0.1:8765 # Linux — install on XFCE python tools/xfce-theme-forge/theme-forge.py init Aurora --preset cursor-blue python tools/xfce-theme-forge/theme-forge.py build configs/Aurora.json python tools/xfce-theme-forge/theme-forge.py install configs/Aurora.json

Presets include nocturn, cursor-blue, midnight-purple, and ember. Outputs land in dist/YourTheme/.

The forge recolors GTK CSS, regenerates xfwm4 borders, and exports a menu pixmap for Whisker Menu.
