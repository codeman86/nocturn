# Nocturn Linux — Website

Marketing / preview site using the same design tokens as the **Nocturn** XFCE theme.

## Preview locally (Windows, Mac, Linux)

```bash
cd website
python -m http.server 8080
```

Open **http://127.0.0.1:8080**

Or double-click `index.html` (some links to `../themes/` work best via a local server).

## GitHub Pages

From repo root, enable Pages with source **/website** or copy this folder to `docs/` if you prefer.
