# Nocturn

XFCE/GTK theme for **Nocturn Linux** — Cursor-inspired dark surfaces with forest green accents.

See the [repository README](../../README.md) for install instructions. The install script also applies the **Nocturn** icon theme and sets the shelter menu icon on Whisker Menu.
